Referral Partner Program SOP

Support SOP | Last updated: February 28, 2026 | Version 1.0

$97/month per referral, recurring. Clients earn passive income as long as their referrals stay active.

Program Overview

After 35 days as a client, they receive automated SMS inviting them to refer other contractors. For each referral that becomes a Slate client, they earn $97/month - recurring for as long as that client stays.

Automation Flows

Flow 7: Review Request (Day 30)

Asks clients to leave a Google review. Includes link to reviews.slatesystems.io. If they reply "REVIEWED", triggers Flow 7.5.

Flow 7.5: Review Coupon Handler

Triggers on "Customer Replied" containing "REVIEWED". Sends $25 discount code (THANKU) for next month.

Flow 8: Referral Partner Nurture (Day 35)

5 SMS over 21 days pitching the referral program. Asks client to reply "YES" if interested.

Flow 8.5: Referral YES Handler

Triggers on "Customer Replied" containing "YES" + has currentclient tag + doesn't have referral-partner-active. Sends referral link, tags as referral-partner-active.

Referral Link

Format: https://refer.slatesystems.io?ref={{contact.id}}

GHL Custom Value: {{custom_values.slate_referral_signup_link}}

Flow

  1. Client shares their unique referral link
  2. Referred person clicks → lands on refer.slatesystems.io
  3. CTA goes to funnel: slatesystems.io/referral-book-now?referrer=CONTACT_ID
  4. They book a call → referrer param captured in GHL
  5. If they close → link referrer in tracking, pay $97/month

When Client Replies "YES"

1
Flow 8.5 auto-triggers

Sends referral link SMS, adds referral-partner-active tag, removes from Flow 8 nurture.

2
Add to Baserow (Manual)

Add partner to Baserow Database 254, Table 952 (Partners). Include: Name, Email, Phone, GHL Contact ID, PayPal Email.

3
Get payment details

Reply asking for PayPal email for monthly payouts.

When Referral Books a Call

1
Check referrer param in GHL

New contact should have referrer field populated with the partner's contact ID.

2
On sales call, confirm

Ask "Who referred you?" to double-check.

3
If they close → Track in Baserow

Add to Table 953 (Referrals): Client name, GHL ID, link to Partner, signup date, status "Active", monthly rate $97.

4
Tag the referrer

Add has-active-referral tag to the partner's GHL contact. Add note: "Referred: [Client Name] - Started [Date]"

Monthly Payout Process (1st of Month)

1
Pull Baserow report

Table 952 (Partners) → filter Status = "Active"

2
For each partner, check referrals

Table 953 (Referrals) → filter by Partner, Status = "Active"

3
Verify in GHL

Confirm each referred client still has currentclient tag (hasn't churned)

4
Calculate & pay

Active referrals × $97 = payout. Send via PayPal.

5
Record payout

Add to Table 954 (Monthly Payouts): Month, Partner, count, amount, status "Paid", date, transaction ID.

6
Notify partner

SMS: "Your referral payout for [Month]: $[Amount] sent! Thanks for spreading the word."

When Referred Client Churns

Stop paying for that referral immediately.
  1. Update Baserow Table 953: Set Status to "Churned", add Churn Date
  2. Next month's payout automatically excludes them
  3. If partner has no more active referrals, remove has-active-referral tag
